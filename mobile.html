Place your app icon here as icon.ico (256x256 recommended). You can create a free .ico file at: https://convertico.com (upload any PNG/JPG and download as .ico) https://favicon.io (also free, supports transparent backgrounds) If you skip this step, Electron will use its default icon. Just delete the icon references in package.json if you don't want a custom icon: Remove: "icon": "build-resources/icon.ico" (appears twice)
